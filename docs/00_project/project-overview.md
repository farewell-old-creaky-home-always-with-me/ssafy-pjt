## 프로젝트 배경

공공 데이터 포털과 지자체 오픈 데이터를 활용해 아파트·다세대 주택 실거래 정보를 누구나 쉽게 조회할 수 있는 서비스가 필요하다. 기존에는 국토교통부 실거래가 공개 시스템에 직접 접속해야 하지만, 검색 인터페이스가 복잡하고 주변 상권·환경 정보와 연계되지 않는다. SSAFY HOME은 이 공백을 채우기 위해 기획됐다.

---

## 프로젝트 목표

- 공공 데이터 기반의 주택 실거래 정보를 지역·조건별로 빠르게 검색할 수 있는 웹 서비스를 제공한다.
- 회원별 관심 지역 저장 기능으로 재방문 사용자의 편의를 높인다.
- 특정 매물에서 목적지까지 A* 알고리즘으로 최적 경로를 제공한다.
- 주택 정보뿐 아니라 주변 상권·환경 정보를 함께 제공해 주거 결정 지원 서비스로 확장한다.

## 대상 사용자

| 사용자 유형 | 설명 |
|-------------|------|
| 일반 방문자 (비회원) | 로그인 없이 주택 거래 정보 검색 및 조회 가능 |
| 회원 | 로그인 후 관심 지역 저장, 마이페이지, 경로 탐색 기능 이용 |
| 관리자 (미정) | 공지사항 작성·수정·삭제 권한 보유 (구현 범위 확정 전) |

---

## 핵심 서비스 가치

1. **정보 접근성**: 국토부 공공 데이터를 정제해 간단한 검색으로 실거래 정보를 제공한다.
2. **맥락 정보 통합**: 주택 정보와 주변 상권·환경 데이터를 하나의 화면에서 확인한다.
3. **경로 탐색**: A* 기반 최적 경로 기능으로 매물과 생활 인프라 간 이동 동선을 시각화한다.
4. **개인화**: 관심 지역 저장으로 반복 방문 없이 원하는 지역 정보를 관리한다.

## 개발 범위

### 필수 기능

- 주택 거래 데이터 수집 및 저장
- 지역별 주택 거래 검색 및 목록 조회
- 주택 상세 정보 조회
- 회원 가입, 로그인, 로그아웃
- 회원 정보 조회·수정·탈퇴
- 관심 지역 등록·조회·삭제
- 공지사항 CRUD

### 추가 기능

- 주변 상권 정보 조회
- 주변 환경 정보 조회
- 지도 API 연동 (매물 위치 시각화)

### 심화 기능

- A* 알고리즘 기반 경로 탐색
- 경로 노드·엣지 그래프 모델링
- 경로 탐색 결과 지도 표시

---

## 범위 외 항목 (Out of Scope)

- 실시간 매물 알림 (푸시 알림, 이메일 알림)
- 매물 직거래 / 중개 연결 기능
- 결제 시스템
- 소셜 로그인 (Google, Kakao 등)
- 모바일 앱
- 관리자 전용 대시보드 (별도 기획 필요)

## 주요 가정 사항

- 주택 거래 데이터는 국토교통부 API를 통해 주기적으로 수집해 DB에 저장한다.
- 주택 거래 데이터 수집은 Spring Batch Job으로 처리한다.
- 행정구역 코드는 VWorld 공공 API를 기준으로 한다.
- 지도 기능은 Kakao Maps API를 연동해 구현한다.
- 인증은 세션 기반으로 구현하며 JWT는 사용하지 않는다.
- A* 경로 탐색에 사용되는 그래프 데이터는 사전에 DB에 구축한다.
- 서울 환경 정보는 서울 열린데이터 광장 API를 활용한다.
