## 도메인 요약

SSAFY HOME은 공공 주택 거래 데이터를 중심으로 주변 상권·환경 정보, 경로 탐색, 회원 관리를 결합한 서비스다. 핵심 도메인은 **주택(House)**, **회원(Member)**, **관심 지역(FavoriteArea)**, **상권(Commercial)**, **환경(Environment)**, **경로(Route)**, **공지사항(Notice)** 7개로 구성된다.

---

## 주요 도메인 개념

### 주택 (House)

- 아파트·다세대 주택 단지 정보를 나타낸다.
- 각 주택 단지는 하나 이상의 거래 이력(HouseDeal)을 가진다.
- 주택은 행정구역 코드(RegionCode)와 연결된다.

### 거래 이력 (HouseDeal)

- 특정 주택의 실거래 정보 1건을 나타낸다.
- 거래금액, 거래일자, 면적, 층 정보를 포함한다.
- 국토교통부 공공 API에서 수집한다.

### 행정구역 (RegionCode)

- 시도, 시군구, 읍면동의 행정구역 코드 체계다.
- VWorld 기준 코드를 사용한다.
- 주택 검색과 관심 지역의 기준 단위다.

### 회원 (Member)

- 서비스 사용자 계정을 나타낸다.
- 이메일·비밀번호 기반 인증을 사용한다.
- 관심 지역 목록을 소유한다.

### 관심 지역 (FavoriteArea)

- 회원이 즐겨찾기로 등록한 특정 행정구역이다.
- 회원 한 명이 여러 관심 지역을 가질 수 있다.

### 상권 (CommercialArea)

- 특정 위치 주변의 업종별 상업 시설 정보다.
- 공공 상권 데이터 API에서 조회한다.

### 환경 정보 (EnvironmentInfo)

- 서울 열린데이터 기반 환경 점검 데이터다.
- 특정 위치 주변의 환경 상태를 나타낸다.

### 경로 노드 (RouteNode)

- A* 경로 탐색에 사용되는 그래프 노드다.
- 위도·경도를 속성으로 가진다.
- 교차로, 주요 지점 등 실제 경로상의 위치를 모델링한다.

### 경로 엣지 (RouteEdge)

- 두 RouteNode를 연결하는 그래프 간선이다.
- 거리(가중치)를 속성으로 가진다.
- 경로 탐색 시 이동 비용을 계산하는 기반 데이터다.

### 공지사항 (Notice)

- 서비스 운영 관련 공지 정보다.
- 제목, 내용, 작성자, 작성일을 포함한다.

## 사용자 역할

| 역할 | 설명 | 주요 기능 |
|------|------|-----------|
| 비회원 | 로그인하지 않은 방문자 | 주택 검색·조회, 공지사항 조회 |
| 회원 | 로그인한 사용자 | 비회원 기능 + 관심 지역 관리, 경로 탐색, 마이페이지 |
| 관리자 | 공지사항 관리 권한 보유자 (미정) | 회원 기능 + 공지사항 작성·수정·삭제 |

## 핵심 비즈니스 흐름

### 1. 주택 거래 데이터 흐름

```
국토교통부 API → 데이터 수집 모듈 → house, house_deal 테이블 → 검색/조회 API → 화면
```

### 2. 관심 지역 흐름

```
회원 로그인 → 행정구역 선택 → 관심 지역 등록 → favorite_area 저장 → 목록 조회
```

### 3. 경로 탐색 흐름

```
매물 상세 화면 → 목적지 입력 → GET /api/routes → RouteService → AStarPathFinder
→ route_node, route_edge 데이터 → 최적 경로 계산 → 경로 결과 반환 → 지도 표시
```

### 4. 주변 정보 흐름

```
매물 위치(위도·경도) → 상권 API 조회 → CommercialArea 데이터 → 화면 표시
매물 위치(위도·경도) → 환경 API 조회 → EnvironmentInfo 데이터 → 화면 표시
```

## 도메인 용어 정의

| 용어 | 정의 |
|------|------|
| 실거래가 | 국토교통부에 신고된 실제 매매 계약 금액 |
| 행정구역 코드 | 시도·시군구·읍면동을 식별하는 공공 코드 |
| 관심 지역 | 회원이 즐겨찾기로 저장한 행정구역 단위 |
| 상권 | 특정 위치 주변의 상업 시설 밀집 구역 |
| 경로 노드 | 그래프 경로 탐색에서 사용하는 지점 단위 |
| 경로 엣지 | 두 노드를 잇는 이동 경로 단위 |
| A* | 휴리스틱 기반 최단 경로 탐색 알고리즘 |

## 도메인 경계

| 도메인 | 외부 의존 | 내부 의존 |
|--------|-----------|-----------|
| House | 국토교통부 API, VWorld API | RegionCode |
| Member | 없음 | 없음 |
| Auth | 없음 | Member |
| FavoriteArea | 없음 | Member, RegionCode |
| Commercial | 공공 상권 API | 없음 |
| Environment | 서울 열린데이터 API | 없음 |
| Route | 없음 (사전 구축 그래프) | RouteNode, RouteEdge |
| Notice | 없음 | Member (작성자 참조) |
